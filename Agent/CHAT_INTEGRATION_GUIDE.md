# 🎮 DyberPet聊天功能使用指南

## 🎉 **新功能：智能聊天窗口**

我们为DyberPet添加了完整的聊天功能！现在你可以通过专门的聊天界面用自然语言控制宠物。

## 🚀 **如何使用**

### **步骤1：启动DyberPet**
```bash
cd D:/project/PirPaw
python run_DyberPet.py
```

**成功启动时会看到：**
```
🎉 Agent系统已成功集成到DyberPet!
💬 现在可以通过聊天窗口控制宠物了:
   • 右键宠物 → 选择'智能聊天'
   • 输入 '让小猫睡觉' 来控制动作
   • 输入 '现在的状态' 查看宠物信息
   • 输入 '走路' 或 '跳舞' 等动作指令
```

### **步骤2：打开聊天窗口**
1. **右键点击桌面宠物**
2. **选择"智能聊天"菜单项**
3. **聊天窗口将弹出**

### **步骤3：开始控制宠物**
在聊天窗口中输入自然语言命令：

## 🎯 **可用命令**

### **🎭 动作控制**
```
让小猫睡觉     → 宠物执行睡觉动作
走路          → 宠物开始走动
站立          → 宠物站立
跳舞          → 宠物跳舞（如果支持）
跳跃          → 宠物跳跃
飞行          → 宠物飞行（特殊宠物）
施法          → 宠物施法（特殊宠物）
```

### **📊 状态查询**
```
现在的状态     → 显示血量、好感度、当前动作
你有什么能力   → 列出可用功能
宠物信息      → 查看宠物详细信息
```

### **🔄 宠物管理**
```
切换到Kitty      → 切换到Kitty宠物
切换到ChrisKitty → 切换到ChrisKitty宠物
切换到TestPet    → 切换到TestPet宠物
```

## 🛠️ **故障排除**

### **❌ 聊天窗口打不开**
```
解决方案：
1. 确保DyberPet正在运行
2. 检查Agent集成是否成功
3. 重新启动DyberPet
```

### **❌ 看不到"智能聊天"菜单**
```
解决方案：
1. 确保启动时看到Agent集成成功信息
2. 检查控制台是否有错误
3. 重新启动DyberPet
```

### **❌ 聊天功能不响应**
```
解决方案：
1. 检查聊天窗口状态栏
2. 尝试重新打开聊天窗口
3. 使用快捷按钮测试
```

### **❌ 动作不执行**
```
解决方案：
1. 查看控制台日志
2. 尝试基础动作（站立、走路）
3. 检查当前宠物是否支持该动作
```
