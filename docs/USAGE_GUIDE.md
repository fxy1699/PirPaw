# PirPaw 使用指南

## 🎯 快速上手

### 1. 启动程序
```bash
python run_DyberPet.py
```

### 2. 基本操作
- **左键拖拽**：移动宠物位置
- **右键点击**：打开菜单
- **双击**：与宠物互动

### 3. 设置配置
1. 右键点击宠物 → 设置
2. 在"基本设置"中配置API密钥
3. 在"自主宠物"中开启AI功能

## 🤖 AI Agent 功能

### 自主宠物模式
- **开启方式**：设置 → 自主宠物 → 启用自主行为
- **行为间隔**：可调整思考间隔（0.1-5分钟）
- **大脑冷却**：可调整大脑冷却时间

### 工具调用
宠物可以调用以下工具：
- **时间查询**：获取当前时间
- **系统状态**：检查系统资源
- **截图功能**：捕获屏幕截图
- **宠物状态**：查看宠物属性
- **梦境生成**：基于情绪生成梦境
- **屏幕分析**：分析屏幕内容
- **工作总结**：生成工作总结
- **提醒功能**：设置和管理提醒

### 日记功能
- **自动记录**：AI自动生成文学性日记
- **情感驱动**：基于宠物情绪生成内容
- **查看方式**：右键 → 日记本

## 🎨 宠物角色

### 默认角色
- **派蒙**：可爱的向导角色
- **Kitty**：经典猫咪角色
- **ChrisKitty**：克里斯猫咪

### 自定义角色
1. 在 `res/role/` 下创建新文件夹
2. 添加动画图片到 `action/` 文件夹
3. 配置 `pet_conf.json` 和 `act_conf.json`
4. 重启程序即可使用

## 📦 背包系统

### 物品类型
- **食物**：恢复饱食度
- **收藏品**：提升好感度
- **特殊物品**：触发特殊效果

### 物品获取
- **完成任务**：番茄钟、专注时间
- **随机掉落**：点击宠物时
- **商店购买**：使用金币购买

## ⏰ 任务系统

### 番茄钟
- **时长**：25分钟工作 + 5分钟休息
- **奖励**：完成可获得物品
- **操作**：右键 → 任务 → 番茄钟

### 专注时间
- **自定义时长**：可设置任意时间
- **暂停功能**：支持中途暂停
- **奖励机制**：按完成比例获得奖励

## 🔧 高级设置

- **通知设置**：控制通知显示

## 🐛 常见问题

### Q: 程序启动失败
**A**: 检查Python环境和依赖安装
```bash
pip install -r requirements.txt
```

### Q: AI功能不工作
**A**: 检查API密钥配置
1. 确保在设置中正确配置了Qwen API密钥
2. 检查网络连接
3. 确认API账户余额充足

### Q: 宠物动画不显示
**A**: 检查资源文件
1. 确认 `res/role/` 下有对应的角色文件夹
2. 检查 `act_conf.json` 配置是否正确
3. 确认动画图片文件存在

### Q: 设置无法保存
**A**: 检查文件权限
1. 确保程序有写入 `settings.json` 的权限
2. 尝试以管理员身份运行
